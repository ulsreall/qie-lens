# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public issue
2. Email: [GitHub Security](https://github.com/ulsreall/qie-lens/security/advisories/new)
3. Include steps to reproduce
4. We'll respond within 48 hours

## Scope

This is a read-only analytics dashboard. It does not handle:
- Private keys or wallets
- User authentication
- Sensitive personal data

The only external dependency is the public QIE Blockscout API.
